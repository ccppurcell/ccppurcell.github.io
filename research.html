Research

My research in structural graph theory relates to characterising families of graph classes. This has implications for computational complexity, because a common approach to hard problems is to restrict the input to some class. We are interested in characterising the family of graph classes which yield an efficient solution to a given problem.

In distributed computing, we consider a set of nodes which communicate with each other in order to solve a problem. Typically, a node can communicate with some subset of the others, giving rise to a graph structure. My main contributions to this area have involved the case when the input graph is a (toroidal) grid.

Recently, I have conducted research in computational social choice. A social choice mechanism can be manipulated if an agent can benefit by reporting a false preference, for example tactical voting. I am interested in the computational complexity of manipulation, particularly from a parameterized complexity point of view.
